# pi-mono (extensions)

This repository now hosts multiple Pi extensions under `extensions/`:

- `extensions/gemini-search` — Gemini subscription-backed web search tool
- `extensions/fetch` — local browser-backed fetch extension (scaffold)

Install each extension package separately by path, for example:

```bash
pi install -l /absolute/path/to/repo/extensions/gemini-search
pi install -l /absolute/path/to/repo/extensions/fetch
```
